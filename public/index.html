Sailfish OS Chum installation

  • Enable "System → Security → Untrusted software → Allow untrusted software" in the device settings
  • Select the required configuration at this page
  • Download Chum RPM with the appeared link under the selection form
  • Tap the downloaded RPM in the device's "File downloaded" notification or in the file browser and confirm the installation
  • Optionally download and install Chum GUI
  • Optionally disable back the "Allow untrusted software"

SailfishOS:Chum installation

  1. Enable "System → Security → Untrusted software → Allow untrusted software" in the Settings app on your Sailfish OS device.
  2. Select the correct configuration for your device at this page.
  3. Then download one of the RPMs via the links which are displayed under the selection form:
    • sailfishos-chum-gui for the SailfishOS:Chum GUI-client
    • sailfishos-chum for a helper RPM, which solely provides an appropriate local repository configuration for utilising SailfishOS:Chum with command line tools as pkcon or zypper
  4. Tap on the downloaded RPM at "File downloaded" notification on your device or in the file browser and confirm the installation.
  5. Preferably disable the "Allow untrusted software" setting again.
